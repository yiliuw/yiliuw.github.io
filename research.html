My research

Cell-type-specific changes in psilocybin-induced psychedelics states, Work in Progress
We propose a new switching linear dynamic system model with efficient inference that reveals cell-type-specific neural dynamics in different brain states.


Uncovering cell type neighborhood changes in Alzheimer’s disease, Work in Progress
We study the cell type distribution changes as the Alzheimer's disease progresses and identify the key biomarkers for the disease. [poster]


Spatial transcriptomic data reveals pure cell types via mosaic hypothesis [paper] [code]
We addressed the long-standing mosaic hypothesis by developing a gold-standard metric for cell-type purity using spatial point processes. [poster (SfN 2024)]


Combinatorial bandits for k-max with value-index feedback [paper] [code]
We developed a novel combinatorial multi-armed bandit algorithm that achieves standard regret bounds despite operating under limited (near full-bandit) feedback. [poster (ICLR 2024)]


Sketching stochastic set utility functions [paper] [code]
We proposed an algorithm that approximates a wide range of stochastic valuation functions with controllable representation size and constant approximation guarantee.
